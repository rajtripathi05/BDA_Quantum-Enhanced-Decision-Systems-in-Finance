# Quantum-Inspired Portfolio Optimization using Big Data Analytics

## 📌 Overview

This project presents a scalable financial decision-making framework that integrates Big Data Analytics with quantum-inspired optimization techniques. The system leverages a Genetic Algorithm (GA) to optimize portfolio allocation by maximizing the Sharpe ratio while handling high-dimensional financial datasets.

## 🚀 Key Features

* Big Data-inspired financial data processing
* Portfolio optimization using Genetic Algorithm
* High-dimensional feature simulation
* Risk-return trade-off using Sharpe Ratio
* Visualization of optimal asset allocation

## 📊 Dataset

The dataset consists of historical stock prices (AAPL, MSFT, TSLA) with additional synthetic features to simulate high-dimensional financial environments.

## ⚙️ Methodology

1. Data preprocessing and cleaning
2. Feature engineering (returns calculation)
3. Covariance and mean return estimation
4. Genetic Algorithm optimization
5. Portfolio evaluation using Sharpe ratio

## 🧬 Genetic Algorithm Workflow

* Initialization of population
* Fitness evaluation (Sharpe ratio)
* Selection
* Crossover
* Mutation
* Iterative optimization

## 📈 Results

* Optimized portfolio allocation across assets
* Improved diversification using synthetic features
* Positive Sharpe ratio indicating better risk-adjusted returns
